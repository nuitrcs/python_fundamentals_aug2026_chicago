# Installation instructions for working on your local computer


## Install Anaconda 

Go to this website: https://www.anaconda.com/download and click the "Get Started" button. Follow the steps to download the installer.

**Mac users**: You want the Graphical Installer. There are different Mac installers if you have an Intel chip or an Apple chip. If you’re not sure which chip your laptop has, click on the apple icon in the top left corner of your screen and choose About This Mac. 

  
**Windows users**: Most Windows computers will require the 64-bit version of Anaconda. Some Windows computers built before mid-2020 are 32-bit. If you have a 32-bit computer you will need the 32-bit version of Anaconda, which you can find by looking at the archives (https://repo.anaconda.com/archive/) for an installer called “Anaconda3-2022.05-Windows-x86.exe”. If you’re not sure which you have, but you are having trouble with the 64-bit version, try the 32-bit version. 

  

If you get Anaconda installed, open the Anaconda Navigator app.  (The Navigator should have a green circle icon).   Open Jupyter Lab (it will open in your web browser, but it doesn’t actually need the internet). Launch a new Python Notebook. If this works, you’re good to go. Note that you may see multiple Jupyter applications in the Navigator – please select Lab. 

  
If you cannot install Anaconda on your computer or if you have too many issues with the installation, don’t worry! You can access the materials online for now via Google Colab, and we can help you later. We will not have time to troubleshoot installation issues during lectures, but there might be time during lunch or at the end of the day. We will talk more about what Anaconda is and what other tools it provides during the workshop. 

  

## Download the Materials 

Go back to the landing page of this GitHub repo (https://github.com/nuitrcs/python_fundamentals_aug2026_chicago/).  Click on the green Code button and choose Download Zip. Unzip the folder on your computer. Open Jupyter Lab through the Anaconda Navigator. In the file tree on the left side of the screen, navigate to the folder you just downloaded and open the part1Lecture.ipynb notebook. 

  
**Notes:** 
1. Jupyter Lab will not let you access external drives, so you will need to save the workshop materials on your local drive. Note that some computers may choose cloud drives (like One Drive or iCloud) as the default download or desktop location.   
2. Please keep all the materials together in the folder you’ve unzipped – do not move single notebooks out of the folder, as they are dependent on other files also contained in the folder.
